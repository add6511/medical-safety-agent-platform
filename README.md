# medical-safety-agent-platform
基层医疗安全型预问诊与随访平台——软件工程课程大作业

## 项目结构

```
medical-safety-agent-platform/
├── backend/          # Spring Boot 后端服务
├── frontend/         # Vue 3 前端（其他成员负责）
├── ai-service/       # Python FastAPI AI服务（其他成员负责）
└── deploy/           # Docker Compose 部署（其他成员负责）
```

## 技术栈

- 后端：Java 17+ / Spring Boot 3.3 / Spring Security / JWT / MySQL / Flyway
- 前端：Vue 3
- AI服务：Python FastAPI
- 部署：Docker Compose

## 项目说明

本项目为教学演示用途，仅使用合成病例数据，不保存真实患者信息，不提供真实诊断和治疗方案。
